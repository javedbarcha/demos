Rubik's Cube 3D

A real-time 3D simulation of a Rubik's Cube with interactive controls and smooth rotation mechanics. The system models cube state transformations and supports algorithm-based solving logic.

  • Developed using Unity3D and C#
  • Implements cube rotation logic and state tracking
  • Optimized for real-time interaction

Climbing 3D

A physics-based 3D climbing simulation that demonstrates character movement across uneven terrain. The system uses raycasting and environmental interaction to dynamically adjust player positioning.

  • Built with Unity3D and physics-based mechanics
  • Implements cliff detection and surface alignment
  • Uses rigidbody-based movement and constraints
