Rubik's Cube 3D – Digital Twin Simulation (Independent Project – Unity3D)

Developed a 3D simulation digital twin replicating real Rubik’s Cube mechanics (2×2, 3×3, and 4×4 cubes).

Key Contributions

  • Designed and modeled the cube system using Unity3D and Blender
  • Implemented cube mechanics and solving algorithms in C#
  • Created an interactive physics-consistent digital twin environment
  • Simulated real-world cube behavior and rotation constraints

Project Overview

  • Realistic cube mechanics (2×2, 3×3, 4×4)
  • Hierarchical object transformations
  • Real-time interaction system

Technical Implementation

  • Raycasting from Screen Space (Screen touch-Drag Input) to Worldspace Cubes Layer selection using Normal and drag vector information
  • Cube state tracking
  • Detection of colors to check for Solve
  • Optimized interaction system

Impact

  • Developed idea, design, and code
  • Published on Google Play Console (~700,000 downloads)
  • Industry collaboration exposure
